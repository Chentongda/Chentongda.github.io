Glance 安装

OpenStack Image 服务是基础设施即服务 (IaaS) 的核心。它接受对磁盘或服务器映像的 API 请求，默认监听9292端口。以及来自最终用户或 OpenStack 计算组件的元数据定义。它还支持在各种存储库类型上存储磁盘或服务器映像，包括 OpenStack 对象存储。

1. 安装与配置 NFS(复用 ha01 主机)

为了保证 Controller 横向扩容，故将 Glance 放在共享存储中；使用NFS功能，将共享目录挂载到管理端
